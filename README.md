# Sistema de Orçamento Imobiliário

Projeto desenvolvido para a disciplina Algorithmic Thinking & Introduction to Object-Oriented Programming.

## Objetivo

Automatizar a geração de orçamentos de aluguel para uma imobiliária, permitindo calcular valores de apartamentos, casas e estúdios de acordo com regras pré-definidas.

## Funcionalidades

* Cálculo de aluguel para apartamentos
* Cálculo de aluguel para casas
* Cálculo de aluguel para estúdios
* Aplicação de descontos
* Parcelamento de contrato imobiliário
* Tratamento de erros de entrada
* Geração de arquivo CSV com as 12 mensalidades

## Tecnologias Utilizadas

* Python
* Programação Orientada a Objetos
* Manipulação de Arquivos CSV
